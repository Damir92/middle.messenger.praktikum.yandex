## Проект - Мессенджер

### Основные команды для работы
- Установка - `npm i`
- Запуск в dev-режиме = `npm run dev` (порт devhost - 3000)
- Запуск локального сервера - `npm start` (порт devhost - 3000)
- Сборка проекта без запуска локального сервера - `npm run build`
